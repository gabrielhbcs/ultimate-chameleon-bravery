# Ultimate Chameleon Bravery

## English

Welcome to **Ultimate Chameleon Bravery**! This web application is a modern reinvention of the classic 'Ultimate Bravery' concept, adapted for the game 'Meccha Chameleon'. It randomly generates poses and colors, providing a fun and unpredictable experience.

### Features

-   **Interactive Display Area**: A prominent section to display current visual content (images or colors).
-   **Visual History Tracking**: Keeps a chronological record of previously viewed items, allowing users to revisit them easily.
-   **Responsive Design**: Optimized for seamless viewing and interaction across various devices and screen sizes.
-   **Dark Mode**: Aesthetically pleasing dark theme applied by default for comfortable viewing.
-   **Language Toggle**: (Based on `style.css`) Allows users to switch between different language preferences for the interface.

### Technologies Used

This project is built using standard web technologies:

-   **HTML5**: For structuring the content.
-   **CSS3**: For styling and layout, including responsive design and custom properties.
-   **JavaScript**: For interactive functionalities and dynamic content management.

## Português

Bem-vindo ao **Ultimate Chameleon Bravery**! Esta aplicação web é uma reinvenção moderna do conceito clássico 'Ultimate Bravery', adaptada para o jogo 'Meccha Chameleon'. Ela gera aleatoriamente poses e cores, proporcionando uma experiência divertida e imprevisível.

### Funcionalidades

-   **Área de Exibição Interativa**: Uma seção proeminente para exibir o conteúdo visual atual (imagens ou cores).
-   **Rastreamento de Histórico Visual**: Mantém um registro cronológico dos itens visualizados anteriormente, permitindo que os usuários os revisitem facilmente.
-   **Design Responsivo**: Otimizado para visualização e interação perfeitas em vários dispositivos e tamanhos de tela.
-   **Modo Escuro**: Tema escuro esteticamente agradável aplicado por padrão para uma visualização confortável.
-   **Alternador de Idioma**: (Baseado em `style.css`) Permite que os usuários alternem entre diferentes preferências de idioma para a interface.

### Tecnologias Utilizadas

Este projeto é construído usando tecnologias web padrão:

-   **HTML5**: Para estruturar o conteúdo.
-   **CSS3**: Para estilização e layout, incluindo design responsivo e propriedades personalizadas.
-   **JavaScript**: Para funcionalidades interativas e gerenciamento dinâmico de conteúdo.
